ShieldScope

macOS dashboard for monitoring iPhone activity over USB — local-first by default.

Privacy posture: collection and analysis stay on your Mac by default — no cloud relay, no external sharing, no telemetry by default.

Logs are stored locally (optional) and are visible to you — export only if you choose.

What it does

Beta

ShieldScope uses the Mac as an observation point while your iPhone is connected via USB, turning signals into a readable timeline.

  • Detect iPhone connect/disconnect sessions
  • Live activity feed + searchable history
  • Creeper Score™ for suspicious-frequency ranking
  • Charts + summaries to spot spikes and patterns

Local-first by design

No data leaves device

ShieldScope is built to provide visibility without becoming spyware itself.

  • No cloud uploads by default
  • User-controlled retention and export
  • Transparent logging: you can see what’s collected
